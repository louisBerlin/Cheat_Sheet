### 1. Dependencies : 

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-data-mongodb</artifactId>
</dependency>
```

### 2. `application.properties` :

```properties
spring.data.mongodb.uri=mongodb+srv://<username>:<password>@<cluster-url>/<database-name>
```
